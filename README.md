# AI-HCI Projects

A collection of AI-powered human–computer interaction experiments using computer vision, speech processing, and emotion detection.

This repository explores multimodal interfaces using:
- MediaPipe
- OpenCV
- DeepFace
- Speech recognition
- Python-based GUIs

Projects include gesture-controlled interfaces, gaze estimation, emotion-aware systems, augmented reality filters, and speech interaction experiments.

Project status: Ongoing
